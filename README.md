Rancher2.0 is an AI-powered farming assistant designed to help farmers manage agricultural activities through a conversational chatbot interface. The system provides guidance on farming tasks such as crop management, harvesting workflows, and agricultural resources. It also includes a simple prototype e-commerce module for managing farming-related products.

Features

AI farming chatbot assistant

Guidance on agricultural tasks

Harvesting workflow suggestions

Prototype agriculture product marketplace

Conversational interface for farm management
